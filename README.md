# Abductor Detection System

A machine learning-based web application that identifies missing persons using facial recognition and similarity matching.

## Features
- Upload images of missing persons
- CNN-based face recognition
- Similarity matching using embeddings
- Police dashboard for case monitoring

## Tech Stack
- Python, Flask
- TensorFlow, OpenCV
- PostgreSQL
- HTML, CSS, JavaScript

## Project Structure
- app/ → Backend and routes
- cnn/ → Model and prediction logic
- templates/ → Frontend pages
- static/ → CSS, JS

## Note
Model files and datasets are excluded due to GitHub size limitations.
